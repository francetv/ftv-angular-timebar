Time bar

Callback:

  • on-cursor-jumped: when cursor jump to antoher position
  • on-cursor-start-dragging: when cursor is clicked and hold by user
  • on-cursor-stop-dragging: when cursor is released by user
  • on-cursor-dragging: when dragged by user

Time bar is 60 sec long
Data:

  • on-cursor-jumped: {{cursorPosition}} sec
  • on-cursor-dragging: {{timeDragged}} sec
  • on-cursor-start-dragging: {{isDragging}}
  • on-cursor-stop-dragging: {{!isDragging}}
